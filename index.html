Copernicus API

A very small API by David Campbell

/tracks

This will get info for a specific track

/tracks/[:trackTitle]

This will give you all the track info. There are only 9

tracks/[:trackTitle]/comments

This will allow you to link to the track comments DB. It expects a POST request

/tracks/[:trackTitle]/comments/[:commentID]

This will allow you to delete a comment with a DELETE request. But you have to have a password.
